# Sistema de Agendamento para Salão de Beleza

## Como executar

1. Certifique-se de ter o Java 21 instalado.
2. No terminal, execute:

   java -jar salao-1.0.0.jar

3. O sistema estará disponível em http://localhost:8080

## Configuração

- O arquivo application.properties pode ser ajustado conforme o ambiente de produção.
- O banco de dados MySQL deve estar disponível e configurado conforme o application.properties.

## Arquivos incluídos
- salao-1.0.0.jar
- application.properties (exemplo)
- pom.xml (referência)
